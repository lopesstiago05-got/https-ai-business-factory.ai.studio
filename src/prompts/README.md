# Repositório de Prompts Dinâmicos (Prompts Registry)

Para garantir flexibilidade, os prompts não são hardcoded. Eles ficam nesta pasta estruturados como templates parametrizáveis (ex: arquivos Markdown `.md` ou JSON), permitindo melhorias de prompt engineering sem alterar a lógica do código principal.

## Estrutura do Módulo:
- `ceo_prompt.md`: Focado em modelagem de negócios, modelo Canvas e tese comercial.
- `research_prompt.md`: Focado em escuta empática de personas e análise de fricções de mercado.
- `product_prompt.md`: Focado em design instrucional e engenharia educacional.
- `marketing_prompt.md`: Focado em estruturas de copywriting comprovadas (AIDA, PAS).
- `supervisor_prompt.md`: Focado em checklist de qualidade, gramática e aderência técnica.
