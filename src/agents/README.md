# Estrutura dos Agentes (Múltiplos Agentes)

Esta pasta conterá as classes especialistas de agentes de Inteligência Artificial.
Cada arquivo implementará um protocolo ou classe base herdada de `BaseAgent`.

## Arquivos Propostos:
- `BaseAgent.ts`: Abstração de inicialização, tokens, consumo de API e log.
- `CEOAgent.ts`: Valida nichos e direciona teses.
- `ResearchAgent.ts`: Crawling sintético e mapeamento de dores da persona.
- `MarketAgent.ts`: Levantamento SEO e análise de palavras-chave.
- `ProductAgent.ts`: Estruturação de sumários e módulos de infoprodutos.
- `WriterAgent.ts`: Geração de conteúdos long-form (longos parágrafos educativos).
- `DesignerAgent.ts`: Geração de paletas cromáticas e engenharia de prompts visuais.
- `MarketingAgent.ts`: Redação de páginas de vendas de alta conversão.
- `PublisherAgent.ts`: Configuração e simulação de empacotamento em plataformas (Ex: Hotmart, Kiwify).
- `FinanceAgent.ts`: Modelos de precificação otimizada baseada em persona.
- `SupervisorAgent.ts`: Auditoria heurística e verificação de selo de qualidade do produto.
