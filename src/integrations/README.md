# Camada de Integrações Externas (Integrations Layer)

O AI Business Factory foi desenhado sob princípios de extensibilidade.

## Integrações Futuras Mapeadas:
- **Plataformas de Distribuição (Webhooks & APIs)**: Kiwify, Hotmart, Monetizze, Eduzz para publicação direta do infoproduto criado pela fábrica.
- **Redes Sociais (Meta, TikTok API)**: Para envio automático de copies e agendamento de posts criativos.
- **E-mail Marketing (Mailchimp, ActiveCampaign)**: Upload direto das sequências de e-mails geradas pelo `Marketing Agent`.
- **WordPress / Webflow (CMS API)**: Para publicação instantânea de Landing Pages ou artigos de blog para SEO.
